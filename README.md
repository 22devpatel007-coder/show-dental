# Dental Clinic Website

A modern, responsive dental clinic website built to help practices establish a strong online presence, showcase their services, and make it easy for patients to connect with the clinic.

This project focuses on clean design, performance, accessibility, and a professional user experience suitable for real-world dental practices.

## Live Demo

https://show-dental.vercel.app/

## Features

- Modern and responsive user interface
- Professional landing page
- About Doctor section

- Image gallery

## Getting Started

### Prerequisites

- Node.js 18 or later
- npm

### Installation

Clone the repository

```bash
git clone https://github.com/22devpatel007-coder/dental-clinic-website-template.git
```

Navigate to the project

```bash
cd dental-clinic-website
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Open your browser

```
http://localhost:5173
```

---

## Production Build

Create a production build

```bash
npm run build
```

Preview the production build

```bash
npm run preview
```
